# C-S-Connet
语言：C#  
框架：Winform  
注：请默认在桌面使用，其他文件位置可以自行修改  
一个简易客户端-服务器，通过Socket通讯，通关缓存实现图片的上传，下载，预览，添加缓存日志以及数据字段查看。
